## ğŸƒâ€â™‚ï¸ Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn
- Supabase account (free tier works!)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/caba-exam-hacker.git
   cd caba-exam-hacker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up Supabase:**
   - Create a free account at [supabase.com](https://supabase.com)
   - Create a new project
   - Go to Project Settings > API
   - Copy your Project URL and anon key

4. **Configure environment:**
   ```bash
   cp .env.example .env
   ```
   Edit `.env` and add your Supabase credentials:
   ```env
   REACT_APP_SUPABASE_URL=your_project_url
   REACT_APP_SUPABASE_ANON_KEY=your_anon_key
   ```

5. **Set up Database:**
   In your Supabase SQL Editor, run:
   ```sql
   -- Create PYQs table
   CREATE TABLE pyqs (
     id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
     subject TEXT NOT NULL,
     question TEXT NOT NULL,
     answer TEXT NOT NULL,
     trick TEXT,
     trick_video_url TEXT,
     year INTEGER,
     exam_type TEXT,
     difficulty TEXT DEFAULT 'Medium',
     created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
   );

   -- Enable Row Level Security
   ALTER TABLE pyqs ENABLE ROW LEVEL SECURITY;

   -- Create policies (adjust as needed)
   CREATE POLICY "Allow public read access" ON pyqs
     FOR SELECT USING (true);
   
   CREATE POLICY "Allow authenticated inserts" ON pyqs
     FOR INSERT WITH CHECK (auth.role() = 'authenticated');
   ```

6. **Start the development server:**
   ```bash
   npm start
   ```

7. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)
